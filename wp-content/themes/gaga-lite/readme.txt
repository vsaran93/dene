Documentation
    Setup Home Page
        -Go to Page->Add New  give the name of page and select home page template and save.
        -Go to Setting->Reading-> A static page (select below) and select page what we vreate for home.
    
    Add Parallax Section
        -Go to Appearance -> Customizer -> Home Page Setting. Enable the section and configure the necessary setting.
